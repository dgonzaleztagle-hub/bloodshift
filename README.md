# BLOOD SHIFT - Prototipo

Prototipo jugable de Blood Shift, un juego indie 2D de acción arcade.

## Instalación

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

El juego se abrirá automáticamente en tu navegador en `http://localhost:3000`

## Controles

- **Flechas**: Mover al personaje
- **Espacio**: Disparar

## Objetivo

Elimina todas las amenazas (enemigos rojos) para completar el turno.

## Tecnologías

- Phaser 3 (Motor de juego)
- Vite (Bundler)
- JavaScript ES6+
